### Requirements

- Nodejs v12.13.x
- NPM 7.1x.x

### Build

- Develop installation

```
1. Download repository.
2. Run command npm install on project root.
3. Run command docker-compose up (or up -d for detached).
4. make request on 5000.
5.
```

- Install dependencies

  ```
  npm install
  ```

- Run locally

  ```
  npm run dev
  ```
